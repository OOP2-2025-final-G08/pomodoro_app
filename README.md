# アプリ名: ポモルーレット

概要:このアプリは、ポモドーロ法(25分間の作業と5分間の短い休憩を1セット)を使って勉強や作業などをすることが可能なアプリです。最初にルーレットを回して何回ポモドーロ法をやるかを決めることができます。


## アピールポイント
ポモドーロ法を利用して作業することができます。また、最初にルーレットを回すことで何セットポモドーロ法で作業するかを決定し、ランダム性を楽しむことができます。一ヶ月分の作業時間を記録し、カレンダーに登録します。また、その一ヶ月の作業時間を日にちごとにグラフで表示することで、自分がどれだけ作業をしたのか一目で分かりやすくしました。
