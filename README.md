# アプリ名: ポモルーレット

概要:このアプリは、ポモドーロ法(25分間の作業と5分間の短い休憩を1セット)を使って勉強や作業などをすることが可能なアプリです。最初にルーレットを回して何回ポモドーロ法をやるかという目標を決めることができます。


## アピールポイント
ポモドーロ法を利用して作業することができます。また、最初にルーレットを回すことで何セットポモドーロ法で作業するかの目標を決定することができ、ランダム性を楽しむことができます。作業(25分)と休憩(5分)の終了をわかりやすくするためにアラーム機能を実装しました。一ヶ月の作業した回数と目標の回数を日付ごとにグラフで表示することで、自分がどれだけ作業をしたのか一目で分かりやすくしました。
